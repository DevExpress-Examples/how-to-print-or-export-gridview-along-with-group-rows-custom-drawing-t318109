Developer Express Example
How to customize the GridControl's print output.

This example demonstrates how to override the default exporting process to take into account a custom drawn content provided via the GridView.CustomDrawFooterCell Event (ms-help://DevExpress.NETv10.1/DevExpress.WindowsForms/DevExpressXtraGridViewsGridGridView_CustomDrawFooterCelltopic.htm)
